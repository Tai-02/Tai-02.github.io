# 🌐 TQT Portfolio Website

Website portfolio cá nhân của **Trần Quốc Tài**

---

## 📁 Cấu trúc file

```
portfolio/
├── index.html        # Trang chủ (Hero + Kỹ năng)
├── about.html        # Giới thiệu bản thân + Skill bars
├── projects.html     # Danh sách dự án
├── contact.html      # Form liên hệ
├── style.css         # CSS dùng chung cho tất cả trang
├── duanweeddetection.jpg   # Ảnh dự án Weed Detection AI
└── Duantrasua.jpg          # Ảnh dự án QLBTS
```

---

## 🚀 Cách chạy

Không cần cài đặt gì thêm. Chỉ cần:

1. Tải toàn bộ file về cùng **1 thư mục**
2. Mở file `index.html` bằng trình duyệt

> **Khuyến nghị:** Dùng **Live Server** trong VS Code để tránh lỗi khi mở file local (Chuột phải vào `index.html` → *Open with Live Server*).

(Có thể xem tại https://tai-02.github.io/)
---

## 📄 Mô tả các trang

| Trang | File | Nội dung |
|-------|------|----------|
| Trang chủ | `index.html` | Giới thiệu ngắn, ảnh avatar, kỹ năng nổi bật |
| Giới thiệu | `about.html` | Thông tin cá nhân, skill bars, thống kê |
| Dự án | `projects.html` | Các dự án đã thực hiện kèm ảnh và link GitHub |
| Liên hệ | `contact.html` | Thông tin liên hệ và form gửi tin nhắn |

---

## 🛠️ Công nghệ sử dụng

- **HTML5** — Cấu trúc trang
- **CSS3** — Giao diện, animation, responsive
- **JavaScript** — Xử lý form liên hệ
- **Font Awesome 6** — Icon
- **Google Fonts (Poppins)** — Font chữ

---

## ✨ Tính năng

- Giao diện dark mode hiện đại
- Avatar với vòng xoay gradient animation
- Hiệu ứng ngôi sao trang trí (hero section)
- Skill bars có phần trăm (trang About)
- Grid dự án 2 cột với hover effect
- Form liên hệ có toast notification
- Navbar sticky với active state
- Responsive layout
